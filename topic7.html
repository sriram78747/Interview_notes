OSI Model

A conceptual framework for how data travels:


  • Physical – Hardware (cables, ports).

  • Data Link – MAC addressing, switching.

  • Network – IP addressing, routing.

  • Transport – Reliable data transfer (TCP).

  • Session – Manages sessions.

  • Presentation – Data translation, encryption.

  • Application – Interfaces with user applications (e.g., browsers).
