Protocols

Protocols are rules that define how data is transferred.


Common Protocols:

  • TCP (Transmission Control Protocol) – Ensures reliable and ordered data delivery

  • IP (Internet Protocol) – Routes data packets to their destination

  • HTTP/HTTPS – Used for web browsing (HTTPS is secure)

  • FTP – Used for transferring files

  • SMTP/IMAP/POP3 – Used for sending and receiving emails

  • DHCP – Assigns IP addresses automatically

  • DNS – Resolves domain names to IP addresses

  • ICMP – Used in diagnostic tools like ping
