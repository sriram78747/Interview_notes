Tools for Troubleshooting

Basic Commands:

  • ping – Checks if a host is reachable

  • tracert – Shows the path packets take to a host

  • ipconfig (Windows) / ifconfig (Linux/macOS) – Shows IP configuration

  • netstat – Displays network connections and ports in use

  • nslookup – Checks DNS resolution

Advanced Tools:

  • Wireshark – Packet sniffer for analyzing network traffic

  • Nmap – Network scanner for discovering hosts and services
