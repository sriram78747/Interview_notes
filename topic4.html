IP Addressing

IP Address

  • A unique string (like 192.168.0.1) that identifies a device on a network.

Private vs Public IP

  • Private: Used within local networks; not routed on the internet

  • Public: Assigned by ISP for global access

Static vs Dynamic IP

  • Static: Manually set; doesn't change

  • Dynamic: Assigned by DHCP; can change over time
